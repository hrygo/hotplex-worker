
  ── Feishu Setup Guide ──────────────────────────

  1. Create App: https://open.feishu.cn/app → "创建企业自建应用"
  2. Add Permissions (权限管理):
     im:message, im:message:send_as_bot, im:resource
  3. Enable Event Subscription (事件订阅):
     im.message.receive_v1
  4. Get Credentials: 凭据与基础信息 → App ID + App Secret

  Docs: https://open.feishu.cn/document/home
