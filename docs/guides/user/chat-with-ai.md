# 与 AI 对话

直接在飞书或 Slack 中给 Bot 发消息，AI 即可实时回复。无需安装任何软件，打开聊天窗口就能用。

---

## 发送消息

给 Bot 发送任意文字，AI 会实时流式回复——

- **Slack**：消息内容逐步更新，你能看到 AI 正在"打字"
- **飞书**：以卡片形式流式展示回复内容

> 直接像跟同事聊天一样发消息就行，AI 会理解你的意图。

---

## 控制命令

### 休眠会话：`/gc` 或 `/park`

暂停当前对话，释放资源，但保留聊天记录。下次发消息会自动恢复。

```
/gc
```

适用于：暂时不用但不想丢失上下文的场景。

### 重置会话：`/reset` 或 `/new`

清空所有上下文，从头开始一次全新对话。

```
/reset
```

适用于：话题完全切换，或对话变得混乱想重新来过。

### 切换工作目录：`/cd`

```
/cd /home/project/my-app
```

AI 会切换到指定目录开始工作。

### 自然语言触发

所有控制命令都可以用 `$` 前缀的中文触发——

| 命令 | 中文触发 | 作用 |
|------|----------|------|
| `/gc` | `$休眠` `$挂起` | 暂停会话 |
| `/reset` | `$重置` | 重新开始 |
| `/cd` | `$切换目录 <路径>` | 切换目录 |

---

## 对话管理

| 命令 | 中文触发 | 说明 |
|------|----------|------|
| `/compact` | `$压缩` | 压缩对话历史，释放上下文空间 |
| `/clear` | `$清空` | 清空当前对话 |
| `/rewind` | `$回退` | 撤销上一轮对话 |
| `/commit` | `$提交` | 让 AI 创建一次 Git 提交 |

---

## 查看状态

| 命令 | 中文触发 | 说明 |
|------|----------|------|
| `/context` | `$上下文` | 查看上下文窗口用了多少 |
| `/skills` | `$技能` | 查看 AI 当前加载了哪些技能 |
| `/mcp` | — | 查看 MCP 服务器连接状态 |

## 调整配置

### 切换模型

```
/model claude-sonnet-4-6
```

或中文触发：`$切换模型`

### 设置权限模式

```
/perm bypassPermissions
```

或中文触发：`$权限模式`

### 调整推理力度

```
/effort high
```

级别越高，AI 思考越深入（但也更慢）。

---

## 权限确认

当 AI 需要执行敏感操作（如修改文件、运行命令）时，会先向你请求批准。

你会收到一条确认消息，需要在 **5 分钟内** 回复。超时未回复将自动拒绝。

---

## 问答交互

AI 有时会主动向你提问并提供选项，直接点击或回复即可。

---

## 语音消息

发送语音消息，AI 会：

1. 自动将语音转录为文字（STT）
2. 根据内容生成回复
3. 回复包含一段语音摘要（TTS）

直接发语音就行，无需额外操作。

---

## 出错重试

如果 AI 偶尔回复出错（如网络波动），系统会自动重试，无需手动操作。你只会看到最终的正确回复。

---

## 帮助

随时输入 `?` 或 `？` 或 `/help` 查看所有可用命令。
